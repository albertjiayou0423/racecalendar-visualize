# 赛道时刻 - 功能概览

## 核心功能

### 1. 实时倒计时（每秒更新）
- ✅ 最近赛事的倒计时精确到**秒**（天 / 时 / 分 / 秒）
- ✅ 北京时间实时显示在顶部
- ✅ 页面自动每秒刷新倒计时数据

### 2. WRC 特殊赛段时间显示
为每个WRC拉力赛精确显示所有特殊赛段：
- **排位测试赛段（Shakedown）**：首日早上
- **第1比赛日、第2比赛日**：中间各日的首个赛段时间
- **决胜赛段（Power Stage）**：末日中午（主赛事）

每个赛段同时显示：
- 当地时间与时区偏移（如 UTC+3）
- 北京时间转换（UTC+8）
- 时间状态标记（"约"表示估计时间）

### 3. 双时区精确显示
所有赛事的所有场次都显示：
- **当地时间**：赛事所在地时区
- **北京时间**：统一北京时间（UTC+8），方便中国观众

### 4. 直播提供商标注
- 🎬 **F1**：五星体育（五星体育直播转播）
- 📺 **WRC**：腾讯视频（WRC 中国大陆转播）
- 🔴 **FE**：官方渠道标注

### 5. 多视图模式
用户可在三种视图之间无缝切换：

#### 📋 列表视图（默认）
- 传统卡片列表，每个赛事一张卡片
- 包含系列、轮次、地点、北京时间、倒计时
- 支持展开查看所有场次详细时间

#### 📅 日期分组视图
- 按赛事主赛事的北京时间日期分组
- 展示每个日期有哪些赛事
- 适合按日期规划观看

#### 📊 统计视图
- 快速统计：总赛事数、即将开始、已结束
- 系列分布：F1 / WRC / FE 各有几站
- 下一场赛事预览
- 完整赛事一览表（按日期和时间排序）

### 6. 高级搜索和过滤
- 🔍 **关键词搜索**：按赛事名称、城市、国家、赛道搜索
- 🌍 **国家快速过滤**：一键选择特定国家的赛事
- 📍 支持系列筛选（全部 / F1 / WRC / FE）
- ⏱️ 支持时间筛选（即将开始 / 全部 / 已结束）

### 7. 赛事提醒系统
- 📌 自动生成标准提醒时间点（已实现后端API）：
  - 赛事前 1 小时
  - 赛事前 30 分钟
  - 赛事前 10 分钟
  - 直播开始时

### 8. 数据源说明
- **F1**：Jolpica F1 API（Ergast 兼容）- 实时获取官方赛程
- **FE**：Formula E 官方 API（pulselive）- 实时获取官方赛程
- **WRC**：FIA/WRC 官方 2026 赛历 - 维护版赛历数据（官方 API 需授权）

### 9. 响应式设计
- 📱 完全移动端适配
- 🖥️ 桌面端优化显示
- 📊 统计面板在小屏幕上隐藏，点击视图切换可访问

## 用户体验亮点

1. **实时性**：倒计时每秒更新，北京时间实时显示
2. **准确性**：双时区显示，无需手动计算
3. **便利性**：一站式查看 WRC / F1 / FE 完整赛程
4. **灵活性**：多种视图和过滤方式满足不同需求
5. **信息完整**：包含直播信息、赛段细节、国家标记等

## 技术实现

### 前端
- React 18 + Next.js 16
- SWR 用于数据获取和缓存
- 每秒自动刷新时间戳
- 动态时区转换（Intl.DateTimeFormat）
- Tailwind CSS 暗色赛车主题

### 后端
- `/api/schedule` 聚合路由
- F1 / FE / WRC 数据源整合
- UTC 到本地时区的动态转换
- 缓存优化（减少 API 调用）

### 工具库
- `lib/format.ts`：格式化、倒计时、时区转换
- `lib/tz.ts`：时区工具、国家代码映射
- `lib/notifications.ts`：提醒系统逻辑
- `lib/wrc-data.ts`：WRC 赛历数据和场次生成
